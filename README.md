# 🏃‍♂️ AI Marathon Coach (AMC)
**〜50代サブ3ランナーの知見 × 科学的理論（VDOT）で、あなたの自己ベスト更新を加速させる〜**

AI Marathon Coach (AMC) は、市民ランナーが効率的かつ安全にフルマラソンの自己ベスト（PB）を更新するために開発された、AI練習メニュー作成ツールです。

## 🌟 AMCの特徴
- **精密な走力診断**：ジャック・ダニエルズ氏のVDOT理論に基づき、小数点単位であなたの走力を分析します。
- **パーソナライズされた12週間メニュー**：AIがあなたの目標、現在の走力、苦手な練習、さらには怪我の状況まで考慮して、世界に一つだけの練習計画を1分で作成します。
- **最新ロジックの融合**：ダニエルズ式の閾値トレーニングに加え、近年注目されている「ノルウェー式」などのエッセンスも取り入れています。
