# AIマラソンコーチ ユーザーマニュアル

## 📖 はじめに

AIマラソンコーチは、ジャック・ダニエルズ博士の「ランニング・フォーミュラ」に基づき、あなた専用のマラソントレーニング計画を自動生成するWebアプリケーションです。

**公開URL**: https://ai-marathon-coach.streamlit.app/

---

## 🚀 使い方

### 1. アプリにアクセス

上記URLにアクセスします。

### 2. 情報を入力

#### 基礎走力
| 項目 | 説明 | 入力例 |
|:-----|:-----|:-------|
| ニックネーム | トレーニング計画で使用される名前 | akira |
| 年齢 | 現在の年齢 | 56 |
| 性別 | 男性/女性/その他 | 男性 |
| 現在のベストタイム | マラソンの自己ベスト | 2:51:30 |
| 目標タイム | 達成したいタイム | 2:48:59 |

#### トレーニング条件
| 項目 | 説明 | 入力例 |
|:-----|:-----|:-------|
| 週間走行距離 | 普段の週間走行距離（km） | 130 |
| 練習可能日数 | 週に練習できる日数 | 6日 |
| ポイント練習 | 週のポイント練習回数 | 3回 |

#### 目標レース
| 項目 | 説明 | 入力例 |
|:-----|:-----|:-------|
| レース名 | 出場するレースの名前 | 東京マラソン |
| レース日 | レースの開催日 | 2026/03/01 |
| 練習レース | 本番前に出場する練習レース | 愛媛マラソン(2/1) |

#### その他
| 項目 | 説明 | 入力例 |
|:-----|:-----|:-------|
| 要望・相談事項 | 持病や練習スタイルの希望 | 土日セット練希望、2部練可能、腰痛の持病あり |

### 3. 計画を生成

「トレーニング計画を生成」ボタンをクリックすると、AIがあなた専用の計画を作成します（1〜2分程度）。

### 4. 結果を確認

- **VDOT計算結果**: 現在のVDOTと目標VDOTが表示されます
- **警告メッセージ**: 条件調整がある場合に表示されます
- **週間トレーニング計画**: 詳細な週間計画が表形式で表示されます

### 5. ダウンロード

「📥 週間トレーニング計画をダウンロード」ボタンでMarkdownファイルをダウンロードできます。

## 💡 自動調整機能

### VDOT差の自動調整
現在のVDOTレベルに応じて、1サイクルで達成可能なVDOT向上幅が自動設定されます。

| 現在のVDOT | 1サイクルの許容差 |
|:-----------|:------------------|
| 〜40（サブ4目標） | 4.0 |
| 40〜50（サブ3:20目標） | 3.0 |
| 50〜55（サブ3:00目標） | 2.5 |
| 55〜60（サブ2:45目標） | 2.0 |
| 60以上（サブ2:35以下） | 1.5 |

目標が許容差を超える場合、**中間目標**が自動設定されます。

### トレーニング条件の自動調整
入力されたトレーニング条件が目標達成に必要な最低条件を満たさない場合、**自動的に最低条件に調整**してトレーニング計画が生成されます。

例：目標サブ2:50で週間走行距離60kmを入力した場合
→ 「週間走行距離: 60km → **100km**」に自動調整

## 💡 用語説明

### VDOT
ジャック・ダニエルズ博士が考案した走力指標。マラソンタイムから算出され、トレーニングペースの基準となります。

### ペースの種類
| 記号 | 名称 | 説明 |
|:-----|:-----|:-----|
| E | Easy | ゆっくりジョグ。会話できるペース |
| M | Marathon | マラソンレースペース |
| T | Threshold | 閾値ペース。1時間程度維持できる強度 |
| I | Interval | インターバル走のペース |
| R | Repetition | 短距離の全力に近いペース |

### 4フェーズ構成
| フェーズ | 目的 |
|:---------|:-----|
| 基礎構築期 | 有酸素能力を高める |
| 強化期 | 持久力を強化する |
| 実践期 | レースペースに慣れる |
| 調整期 | 疲労を抜いてピークに持っていく |

### 練習スタイル
| 用語 | 説明 |
|:-----|:-----|
| 2部練 | Eペースジョグを午前と午後に分けて行う |
| セット練 | 土日連続でポイント練習を行う |
| ダブルスレッショルド | 午前と午後に閾値走を行う（要明示的希望） |

---

## ⚠️ 注意事項

- **2部練とダブルスレッショルドの違い**: 
  - 2部練: Eペースジョグを分割（腰への負担軽減に有効）
  - ダブルスレッショルド: 高強度練習を2回（明示的に希望した場合のみ）

- **レースまで12週未満の場合**: 12週間の計画が生成され、開始日が過去になります。現在の週から始めてください。
